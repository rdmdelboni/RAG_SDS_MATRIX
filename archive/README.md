# Archive Directory

This directory contains historical documentation and unused code that has been archived to keep the project root clean and organized.

## Structure

### `implementation_notes/`
Implementation summaries and technical notes from various feature additions:
- CAMEO integration
- PubChem enrichment
- Editable table components
- Review tab implementation
- Column resize features

### `session_notes/`
Session summaries, completion reports, and progress tracking documents:
- Session summaries
- Completion reports
- Testing results
- Bug fix reports
- Enhancement tracking

### `old_scripts/`
Deprecated scripts and test files:
- `migrate_db.py` - Old database migration script (no longer needed)
- `INTEGRATION_EXAMPLE.py` - Example integration code (reference only)
- `test_*.py` - Old test drafts (actual tests are in `/tests/`)
- `colors.html` - Unused HTML color palette
- `package-lock.json` - Empty Node.js lock file (not needed for Python project)

## Important Notes

- **Do not delete these files** - They contain valuable historical context and implementation details
- **These files are not part of the active codebase** - The application does not depend on them
- **For active documentation**, see:
  - `/README.md` - Main project documentation
  - `/docs/` - Current user guides and technical documentation
  - `/TODO.md` - Current project tasks
